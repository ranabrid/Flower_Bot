#### **UC-01: Установить первое напоминание**
<br>

**Цель:** Установить первое напоминание о покупке цветов по заранее установленному расписанию 
<br>

**Актор:** Пользователь
<br>

**Предусловие:**
- бот запущен на сервере;
- пользователь имеет активный аккаунт в Telegram
<br>


**Триггер:** Пользователь отправляет команду '/start'
<br>
**Основной сценарий:**

1. Система отправляет приветственное сообщение и предложение начать настройку.
2. Пользователь нажимает «Да, пожалуйста».
3. Система сохраняет id пользователя в базу данных.
4. Система планирует первое напоминание на заданную дату.
5. Система подтверждает успешное планирование.
<br>

**Альтернативный сценарий (2а Пользователь передумал):**

2а.1 На шаге 2 пользователь нажимает «В другой раз».
2а.2 Система отправляет сообщение: «Хорошо, напишите /start, когда передумаете».
2а.3 Система завершает диалог.
<br>

**Ограничения:**

- бот может планировать задачи только на основе текущего часового пояса (Europe/Moscow);
- бот хранит только chat\_id и next\_remind\_date, не собирая персональные данные.
<br>

**Исключения:**

Если соединение с БД потеряно, система логирует ошибку
<br>

**Минимальные гарантии:**

Система гарантирует, что при любом исходе данные пользователя не будут повреждены (не будет создано дубликатов в БД).

<br>

**Гарантии успеха:**

- в БД создана запись с актуальным chat\_id;
- пользователь получил подтверждение о выполненном действии.
